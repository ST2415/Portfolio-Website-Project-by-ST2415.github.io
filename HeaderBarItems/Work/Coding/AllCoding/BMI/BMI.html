Progress

1.  สร้างกรอบครอบเนื้อหา

2.  เกริ่นนำข้อมูลเบื้องต้น

3.  สร้างกล่องรับข้อมูลส่วนสูงและน้ำหนัก

4.  สร้างปุ่มพร้อมเรียกใช้ฟังก์ชันจาก Java Script

5.  เมื่อคำนวณเสร็จ ข้อมูลจะแสดงผลออกมาในกล่องนี้

6.  เรียกใช้ฟังก์ชัน Java Script จากชื่อนี้

7.  สร้างตัวแปรรับค่าของข้อมูลส่วนสูงและน้ำหนักจาก id ของข้อมูล

8.  สร้างตัวแปรเป็นผลการคำนวณ

9.  เรียกใช้ฟังก์ชันแสดงผลการคำนวณ

10.  รับข้อมูลส่วนสูงและน้ำหนักมาคำนวณ แล้วส่งข้อมูลกลับเป็นทศนิยม 2 ตำแหน่ง

    10.1  เช็คว่า หากข้อมูลเป็นจำนวนเต็มให้นำมาหาร 100 เพื่อทำเป็นทศนิยมก่อน
       แล้วจึงนำไปคำนวณ
    10.2  ถ้าเกิดว่าข้อมูลไม่ได้ถูกกรอก จะเปลี่ยนเป็น 0

11.  นำค่า BMI ที่ได้มาจัดตามเกณฑ์

12.  สร้างฟังก์ชันส่งข้อมูลกลับไปยังหน้าเว็บไซต์
