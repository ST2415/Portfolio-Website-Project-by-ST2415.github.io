Hamster Village

Hamster Village  เป็นเกมที่ผู้เล่นได้รับบทบาทในการป้องกันเมืองจากศัตรู  ในสองรอบแรกผู้เล่นจะต้องไม่ให้ศัตรูโจมตีเมืองได้  เมื่อถึงตาที่  3  ผู้เล่นจะต้องกำจัดบอสโดยไม่ต้องป้องกันเมืองแล้ว

Genre : Coding, Game

Program : VSCode, Unity

Technic : C#

My role : Artist (UI Coding)

Result

Progress

Music

1.  เป็นการสร้างตัวแปรเพื่อเก็บ AudioMixer ในการปรับแต่งเสียงในเกมเอาไว้

2.  ใช้ [SerializeField] เพื่อช่วยในการเก็บข้อมูลของเสียงเอาไว้

3.  จากนั้นทำฟังก์ชันรับข้อมูล สำหรับตั้งค่าเสียงต่างๆ โดยใช้สูตรการคำนวณเดซิเบล

Change Scene

1.  สร้างตัวแปรเลขของฉากและคลาสแม่แบบส่วนรวม(ใช้กับโค้ดอื่นๆได้)

2.  สร้างฟังก์ชันในการเปลี่ยนฉาก โดยเปลี่ยนฉากไปตามลำดับที่เลือก

3.  สร้างฟังก์ชันออกจากเกม โดยใช้คำสั่ง Application.Quit();

UI - Enemy Remaining

1.  เป็นการสร้างตัวแปรแบบส่วนตัวเพื่อเก็บ Text บนหน้า UI และจำนวนของศัตรู

2.  เมื่อเริ่มเกม ศัตรูจะมีทั้งหมด 20 ตัว แล้วจะทำการ update ข้อมูลทุกๆวินาที

3.  สร้างฟังก์ชันในการแสดงศัตรูที่เหลืออยู่ในเกม บนหน้า UI

UI - Arrow Left

1.  สร้างตัวแปรแบบส่วนตัวในการเก็บจำนวนของลูกธนู และ Text ส่วนของ UI ไว้

2.  สร้างฟังก์ชันแสดงข้อมูลของลูกธนูบนหน้าจอ และจะ update ข้อมูลทุกๆวินาที

UI - Scoring

1.  กำหนดตัวแปรสำหรับส่วนของการแสดงผลหน้า UI และข้อมูลของคะแนน

2.  เมื่อเริ่มเกมคะแนนจะปรับเป็น 0 และจะเรียกใช้ฟังก์ชันในการเช็คและแสดงคะแนนทุกๆวินาที

3.  สร้างฟังก์ชันในการแสดงคะแนน

4.  สร้างฟังก์ชันในการเช็คคะแนน ถ้าคะแนนเพิ่มจะทำการเปลี่ยนคะแนนใหม่

5.  สร้างฟังก์ชันในการรีเซ็ตคะแนน

UI - Count down and Wave

1.  สร้างตัวแปรในการเก็บเวลา รอบ และสถานะ

2.  ระบบจะ update ระบบเวลาและรอบทุกๆวินาที

3.  หากเวลาเท่ากับ 0 จะทำการเริ่มรอบใหม่และให้เวลาพักระหว่างรอบสักครู่

4.  หากเวลาช่วงพักระหว่างเปลี่ยนรอบหมดลง จะทำการเริ่มรอบใหม่

5.  หากจำนวนรอบในปัจจุบันเท่ากับ 4 จะทำการเปลี่ยนฉากไปหน้าสรุปผล

6.  สร้างฟังก์ชันในการแสดงรอบ

7.  สร้างฟังก์ชันในเปลี่ยนรอบ

8.  สร้างฟังก์ชันในการเริ่มรอบ โดยเวลาจะลดลงและแสดงบนหน้า UI หากเวลาเหลือ 10 วินาที จะเปลี่ยนเป็น Text สีแดง

9.  สร้างฟังก์ชันในการพักระหว่างรอเริ่มรอบใหม่ โดยเวลาจะแสดงเป็นสีเขียว หากเวลาเท่ากับ 0 จะทำการเรื่มรอบใหม่

10.  สร้างฟังก์ชันในการส่งเลขรอบบนหน้าจอใหม่โดยให้รอเวลาพักหมดลง

11.  สร้างฟังก์ชันส่งเลขในการเปลี่ยนรอบให้เพิ่มขึ้นมา 1

UI - HP

1.  สร้างตัวแปรในการเก็บเลือดของเมือง ผู้เล่น สี ฉาก และเวลา

2.  สร้างฟังก์ชันเลือดในรูปแบบเริ่มต้นและรูปแบบปัจจุบันของเมือง หากเลือดลดเหลือตามช่วงใดของค่าสี สีก็จะเปลี่ยนทันที

3.  สร้างฟังก์ชันเลือดในรูปแบบเริ่มต้นและรูปแบบปัจจุบันของผู้เล่น หากเลือดลดเหลือตามช่วงใดของค่าสี สีก็จะเปลี่ยนทันที

4.  เมื่อเริ่มเกม ผู้เล่นต้องปกป้องเมืองโดยเลือดของเมืองจะถูกแสดงเป็นหลัก

5.  ในระหว่างเล่น เกมจะ update เลือดทุกๆวินาที

6.  ขณะเล่น ถ้าเกิดเลือดน้อยกว่าหรือเท่ากับ 0 จะทำการเปลี่ยนไปฉากจบทันที

7.  ขณะเล่น ถ้าเปลี่ยนเป็นรอบที่ 3 เลือดจะผู้สลับเป็นของผู้เล่น และหากผู้เล่นตาย ก็จะทำการเปลี่ยนไปฉากจบทันที

UI - Pause Menu

1.  สร้างตัวแปรในการเก็บหน้าต่างเมนู สถานะ เละฉาก

2.  เริ่มเกมจะดำเนินตามปกติ เมื่อผู้เล่นกดปุ่ม Escape จะเรียกใช้เงื่อนไขดังนี้

    2.1  หากค่าสถานะเป็น จริง ระบบจะสั่งใช้ฟังก์ชันกลับไปเป็นเหมือนเดิม
       และจะปิดไม่ให้มองเห็น Cursor Mouse ได้
    2.2  หากค่าสถานะเป็น เท็จ ระบบจะสั่งใช้ฟังก์ชันหยุด และจะเปิดให้มองเห็น Cursor Mouse ได้

3.  สร้างฟังก์ชันหยุดเกมชั่วคราว

4.  สร้างฟังก์ชันหเล่นเกมต่อ

5.  สร้างฟังก์ชันเปิดหน้าต่างปรับแต่งเสียง

6.  สร้างฟังก์ชันออกจากหน้าปรับเสียงเพื่อกลับไปหน้าหยุดเกม

7.  สร้างฟังก์ชันออกจากหน้าปรับเสียงเพื่อเล่นเกมต่อ
