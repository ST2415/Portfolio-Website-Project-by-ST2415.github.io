Defused the Bomb

เกมแก้ระเบิดผ่านการทายตัวเลข  โดยสามารถกำหนดจำนวนตัวเลขทั้งหมด  จำนวนระเบิด  และจำนวนรอบในการทายระเบิดได้

Genre : Coding, Game

Program : Google Colab

Technic : Python

Result

Progress

1.  อันดับแรก เราจะทำการ import random มาเพื่อใช้ในการสุ่มเลขของระเบิด
2.  สร้างฟังก์ชันเกมขึ้นมา จากนั้นออกแบบข้อความการโต้ตอบกับผู้เล่น
3.  สร้างตัวแปรในการรับข้อมูล นั่นก็คือจำนวนตัวเลขทั้งหมด จำนวนระเบิด และจำนวนรอบในการทายระเบิด

4.  เปลี่ยนประเภทตัวแปรที่รับมาให้เป็นจำนวนเต็ม และสร้างตัวแปรในการเก็บคะแนน
5.  สร้างตัวแปรขึ้นมา ไว้สำหรับการเก็บระเบิด โดยจะทำการสุ่มระเบิดตั้งแต่  1  ไปจนถึง ตัวเลขที่กรอก

6.  เมื่อเริ่มเกม เราจะทำการสร้างเงื่อนไขมาทั้งหมด 3 เงื่อนไข ในการกำหนดสถานะของเกมคือ ชนะ แพ้ และกำลังดำเนินเกมอยู่  (โดยเกมจะวนซ้ำไปเรื่อยๆจนกว่าจะเข้าเงื่อไขใดเงื่อนไขหนึ่ง)
7.  ถ้ามีคะแนนเท่ากับจำนวนระเบิด เป็นเงื่อนไขในการชนะ
8.  ถ้าจำนวนการทายเท่ากับ  0  เป็นเงื่อนไขในการแพ้

9.  ถ้ามีคะแนนไม่เท่ากับจำนวนระเบิด เป็นเงื่อนไขกำลังดำเนินเกมอยู่

    9.1  สร้างตัวแปรเพื่อรับตัวเลขของระเบิดที่ผู้เล่นทาย และแปลงให้เป็นตัวเลข
    9.2  สร้าง for loop ขึ้นมาเพื่อดึงค่าแต่ละตัวของระเบิดมาเช็ค
    9.3  ถ้าผู้เล่นทายระเบิดถูก จะบวกคะแนนเพิ่ม ลดจำนวนการทาย และลบระเบิดที่ทายไปแล้วออก
       จากนั้นก็บอกผลลัพธ์การทายกับผู้เล่น
    9.4  ถ้าผู้เล่นทายระเบิดผิด จะลดจำนวนการทาย และบอกผลลัพธ์การทายกับผู้เล่น

10. เมื่อทำเสร็จแล้ว ก็สามารถเรียกใช้ฟังก์ชันตัวเกมได้เลย
