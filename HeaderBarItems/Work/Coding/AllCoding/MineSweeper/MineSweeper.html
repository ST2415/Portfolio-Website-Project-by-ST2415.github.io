Mine Sweeper

เกมวัดดวงสุ่มขุดเหมืองแร่หลบระเบิด  หากเราหลบได้  เลขที่อยู่รอบตัวเราจะบอกใบ้จำนวนระเบิดแถวนั้น  เกมนี้เป็นเกมแรกที่หนูได้รับมอบหมายให้ช่วยกันกับทีม  ค้นหาข้อมูลของเกมนี้จากเน็ตเพื่อที่จะนำมาทำความเข้าใจการทำงานของโค้ด

Genre : Coding, Game

Program : Google Colab

Technic : Python

My role : Resercher

Result

Progress

1.  อันดับแรก เราจะทำการ import random มาเพื่อใช้ในการสุ่มจุดที่ระเบิดอยู่

2.  สร้างฟังก์ชันเช็คระเบิดแผนที่รอบๆตัว กำหนดให้ n คือขนาดของแผนที่(5x5) และ k คือจำนวนระเบิด

3.  จากนั้นทำการสร้างแผนที่ขึ้นมาขนาด 5x5 และใส่ 0 ลงไปทุกช่อง (ยังไม่มีการใส่ระเบิด)

4.  ใน for loop ถัดไปจะเป็นการเริ่มใส่ระเบิดลงไปตามจำนวนที่กำหนดไว้(10 ลูก) แทนระเบิดด้วยตัว X

5.  ต่อไปคือการเช็คว่า หากมีระเบิดอยู่ในแปดทิศนี้จะแสดงจำนวนระเบิดที่อยู่รอบๆ

6.  เมื่อเช็คเสร็จแล้วจะทำการส่งข้อมูลมาเขียนทับบนแผนที่

7.  PlayerMap จะทำหน้าที่ซ่อนข้อมูลไว้ ไม่ให้ผู้เล่นเห็นตำแหน่งต่างๆขณะเล่น

8.  DisplayMap เป็นหน้าตาเกมในปัจจุบันที่ผู้เล่นเห็นเมื่อเริ่มเกมไปแล้ว

9.  DisplayWon เป็นฟังก์ชันในการเช็คแต่ละแถว หากทำการเปิดครบทุกช่องโดยไม่โดนระเบิด ผู้เล่นจะชนะทันที

10.  เป็นฟังก์ชันแสดงคะแนนและกล่าวอำลากับผู้เล่น

11.  กำหนดตัวแปรต่างๆในเกม

12.  หากผู้เล่นยังไม่ชนะ เกมจะดำเนินการต่อไปเรื่อยๆ โดยจะมีช่องให้ใส่แกน x(แนวนอน) กับ y(แนวตั้ง)

13.  ถ้าผู้เล่นทายโดนระเบิด เกมจะปรับแพ้ แล้วเฉลยกระดารแผนที่และทำการแจ้งคะแนนทันที

14.  ถ้าผู้เล่นเปิดไม่เจอระเบิด เกมจะยังดำเนินต่อไปเพื่อสะสมคะแนนในการหลบระเบิดต่อไป

15.  แต่ถ้าหากผู้เล่นไม่โดนระเบิดเลย เกมจะปรับให้ชนะทันทีพร้อมแจ้งคะแนน

16.  ทำการเรียกใช้งานฟังก์ชั่นเกม เพื่อเริ่มเล่น
