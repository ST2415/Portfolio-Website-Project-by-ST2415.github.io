Blooko

หนังสืออ่านจนเบื่อแล้ว  จะทิ้งก็นึกเสียดาย  เว็บไซต์  Blooko  เปิดพื้นที่ให้ทุกคนสามารถเข้ามาแลกเปลี่ยนหนังสือได้  โดยจะเปิดโอกาสให้ผู้รับและผู้ส่งได้พูดคุย  ตกลงรับหนังสือได้ตามสะดวกไม่ว่าจะเป็นการนัดเจอกันหรือส่งไปทางไปรษณีย์

Genre : UX/UI Design, Figma

Scope : 1 month and 1 week

Size : 1728px x 1117px, 390px x 844px

Idea and Reference

Progress

(ไม่ได้เก็บภาพระหว่างการทำงานไว้ค่ะ)

1.  ทำการตั้งโจทย์ขึ้นมาก่อน โดยใช้เว็บ UI Coach

2.  รวบรวมข้อมูลเกี่ยวกับเว็บไซต์ที่มีการบริการให้ทุกคนแลกเปลี่ยนส่งของกัน

3.  ร่างโครงร่างหน้าตาแต่ละหน้า

4.  รวบรวมข้อมูลรูปต่างๆสำหรับการทำตัวอย่าง

5.  เริ่มออกแบบหน้าแรก โดยเริ่มจาก Navbar และ Logo ที่จะเป็นตัวพาเราไปยังส่วนต่างๆของเว็บไซต์

6.  จากนั้นก็ออกแบบส่วนของ Header ที่จะเป็นจุดที่ทำให้ผู้อ่านโฟกัสเป็นสิ่งแรก พร้อมเขียนคำอธิบายและชวนเชิญสั้นๆ

7.  ส่วนของตัวเนื้อหาเว็บไซต์จะประกอบด้วย 2 ส่วนคือ การเสนอข้อดีของเว็บและหนังสือ

8.  ส่วนสุดท้ายคือ Footer เป็นที่สำหรับข้อมูลของเว็บหรือช่องการติดต่อ

9.  หน้านี้จะรวมหนังสือทุกประเภทเข้าไว้ด้วยกัน

10.  หน้าข้อมูลหนังสือจะมีข้อมูลต่างๆเก็บไว้ รวมถึงจำนวนครั้งในการแลกเปลี่ยนเพื่อเป็นการประกอบการตัดสินใจในการเลือก

11.  หากต้องการเก็บหนังสือไว้เป็นตัวเลือก ก็สามารถกดปุ่มรูปดาวได้เพื่อทำการบันทึก

12.  เมื่อกดปุ่มแลกเปลี่ยน ระบบจะพาไปยังหน้าแชทของผู้โพสต์เพื่อตกลงการรับหนังสือกันเอง

13.  หน้านี้จะเป็นการสมัครสมาชิกและเข้าสู่ระบบ
