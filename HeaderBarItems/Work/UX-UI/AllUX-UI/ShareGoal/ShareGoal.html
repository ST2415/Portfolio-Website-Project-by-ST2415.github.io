ShareGoal

แอป  ShareGoal  เป็นแอปที่คุณสามารถแบ่งปันเรื่องราวของคุณ  ไม่ว่าจะเป็นความฝัน  เป้าหมาย  หรือภารกิจให้ผู้อื่นอ่านหรือแลกเปลี่ยนความคิดเห็นและช่วยแนะนำก็ได้เช่นกัน  มาเริ่มเขียนเป้าหมายของคุณในพื้นที่ที่พร้อมจะสนับสนุนคุณกันเลย !

Genre : UX/UI Design, Figma

Scope : 3 weeks

Size : 390px x 844px

Result

Idea and Reference

Progress

1.  ทำการตั้งโจทย์ขึ้นมา โดยใช้เว็บ UI Coach

2.  กำหนดองค์ประกอบ ขอบเขตของงานและตั้งกลุ่มเป้าหมาย

3.  รวบรวมข้อมูลจากการทดลองใช้แอปที่เกี่ยวข้องและสนใจในการดีไซน์

4.  ร่างโครงร่างหน้าตาแต่ละหน้า

5.  เริ่มออกแบบหน้าแรก ซึ่งครั้งนี้หน้าบัญชีผู้ใช้จะอยู่หน้าแรก

6.  ข้อมูลสถิติและเป้าหมายจะอยู่ถัดจากบัญชีผู้ใช้ เพื่อความสะดวกในการดูข้อมูล

7.  เมื่อกดปุ่มรูปบวกจะเป็นการเพิ่มเป้าหมาย

8.  หากลองเลื่อนเป้าหมายที่โพสต์ไว้ จะมีปุ่มสำหรับแก้ไข เปลี่ยนความเป็นส่วนตัวและรายงาน

9.  หน้า Social ผู้ใช้จะสามารถมองเห็นการโพสต์ของทุกคนได้

10.  การแจ้งเตือนจะถูกส่งก็ต่อเมื่อมีคนมีปฏิสัมพันธ์กับผู้ใช้หรือผู้ใช้ได้ทำการติดตามผู้ใช้รายอื่นหรือโพสต์

11.  หน้าตั้งค่า ผู้ใช้สามารถปรับตั้งค่าข้อมูลส่วนตัวและธีมได้ตามสะดวก
